############### INSTRUCTIONS ##################

Compile simply by:

cd qcv-code
mkdir build
cd build
cmake ..
make

and

cd qcv-mods
mkdir build
cd build
cmake ..
make

Binaries and a copy of the parameter files for qcv-mods will be placed in qcv-mods/build/bin
