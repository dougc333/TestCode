Instructions to use the New College Data set (hhttp://www.robots.ox.ac.uk/NewCollegeData/) with QCV:

 1.- Download data set and rectifier tool

 2.- Rectify images

 3.- Edit sequence_newcollege.txt and adapt the path to the downloaded data set

 4.- Adapt and copy the file camera.xml to the directory where the images are contained

 5.- run: ./stereoSFM params_stereoSFM/newcollege/sequence_newcollege.xml params_stereoSFM/newcollege/params_stereoSFM.xml
